# <img src="https://cdn.jsdelivr.net/gh/chtof/chocolatey-packages/automatic/aethersdr/aethersdr.png" width="48" height="48"/> [AetherSDR](https://chocolatey.org/packages/aethersdr)

AetherSDR brings full FlexRadio operation to Linux, macOS, and Windows — each a native build, no Wine or virtual machines. A native aarch64 build also runs on Raspberry Pi and other embedded ARM devices. Built from the ground up with Qt6 and C++20, it speaks the SmartSDR protocol natively and aims to replicate the full SmartSDR experience.
## Highlights
- GPU-accelerated spectrum & waterfall
- Multi-slice & multi-panadapter
- KiwiSDR public-receiver browser
- Aetherial Audio Channel Strip
- Six client-side noise-reduction engines
- DAX virtual audio + IQ
- AetherModem packet radio
- AetherSweep
- SpotHub
- CW operator suite
- FreeDV RADE
- SmartLink remote + TCI v2.0 server
- Broad hardware control
