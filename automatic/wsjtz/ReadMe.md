# <img src="https://cdn.jsdelivr.net/gh/chtof/chocolatey-packages/automatic/wsjtz/wsjtz.png" width="48" height="48"/> [WSJT-Z](https://chocolatey.org/packages/wsjtz)

WSJT-Z is a fork of [WSJT-X|https://wsjtx.github.io/wsjtx/index.html] by **Joe Taylor K1JT** and the WSJT Development Group. Initially developed as an automation project, WSJT-Z now focuses on extending the operating workflow with deeper filtering, smarter automation, and a more responsive UI — while keeping full compatibility with the upstream WSJT-X protocols.

WSJT-Z supports **FT8, FT4, FT2, JT4 / JT9 / JT65, Q65, MSK144, WSPR, Echo and FreqCal**.

?? Always monitor your transceiver while using WSJT-Z, unless unattended/automated operation is explicitly permitted by the regulations in your country.
